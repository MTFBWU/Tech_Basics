Estimated Production Process

Step 1: Wire the sensor to the arduino board

Step 2: Testing the communication

- Make sure arduino can communicate with the sensor

Step 3: Data Retrieving

- Test the changes of X & Y Axis (getX() and getY())

- Use Serial library to send data through USB cable or any wireless data module, like a bluetooth or a 433MHz radio, attached to serial interface.

Step 4: 3D Modeling

- print and assemble sensor into the phonecase
