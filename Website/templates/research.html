what is a mouse, types of mouses and how a mouse works.

What is a mouse?

A computer mouse is a hand-held pointing device that detects two-dimensional motion relative to a surface. This motion is typically translated into the motion of a pointer on a display, which allows a smooth control of the graphical user interface

Types of mouse:

  • Wireless Mouse
  • Trackball Mouse
  • Optical Mouse
  • Laser Mouse
  • Ball Mouse
  • Optical-mechanical Mouse

We will be using a Optical mouse sensor for this project. This is because Optical is far cheaper while being wireless and functions based on the same theory as laser mouse while with more stable results.

How does an Optical mouse function?

Optical mice contain LEDs. ... Able to work on almost any surface, the mouse has a small, red light-emitting diode (LED) that bounces light off that surface onto a complementary metal-oxide semiconductor (CMOS) sensor. The CMOS sensor sends each image to a digital signal processor (DSP) for analysis.

in other words...

--> Optical mouse is basically a low-resolution camera.

--> It combined with 2 lenses and a source of illumination that tracks the peripheral’s X and Y coordinates thousands of times per second.

--> The CMOS sensor collects the light, and converts the light particles into an electrical current. This analog data is then converted into 1’s and 0’s,
